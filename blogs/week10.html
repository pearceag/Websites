This week outside of class, I looked into how to create Dark Mode on a page by pressing a button. I used both Javascript and CSS to complete this task. Below is the code for the Javascript that triggers dark-mode when pressed: 

function myFunction() {
    var element = document.body;
    element.classList.toggle("dark-mode");
}
Then in my CSS file I had to define what dark-mode looked like. I needed to change the background color to a darker color and change the text to white. See the code below: 
.dark-mode {
    background-color: #292929;
    color: white;
    }
