Outside of class, I have researched how to use the filter property in css. The filter function can do so much and is a great way to make a website look more professional. The filter property has many functions such as blur() which blurs an image, brightness() which allows the user to adjust the brightness of an image, and constrast() which adjusts the constract of an image, inverted() inverts the colors in the image, saturated() which adds more emphasis on the colors already present in an image, and opacity() which fades the image the higher the percentage passed in is.
