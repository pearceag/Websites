Since the Title Tooltip was so easy to implement I also looked up how to make text scroll in CSS. Scrolling text can be very useful on websites to catch the eye of users and great for advertising on websites. What's cool about this feature is you can make it scroll from right to left, left to right, up to down, or down to up. You can determine the direction of the scrolling by modifying the transform translateX to scroll horizontally on the x-axis and transform translateY to scroll vertically on the y-axis. To make the text scroll from left to right you pass in a positive percentage in as the parameter for translateX and a negative percentage value to scroll from right to left. Likewise, to make text scroll down to up, you would pass a positive percentage in as the parameter for translateY and a negative percentage to scroll from up to down. Below is my implementation of the scroll animation in CSS. Do you recognize the sentence being scrolled? If not it's the first sentence Harry Potter and the Sorcerer's Stone (I am a HUGE Harry Potter Fan :).

"The hottest day of the summer so far was drawing to a close and a drowsy silence lay over the large, square houses of Privet Drive.”

Below, is the HTML and CSS code for the feature above.

<div id="scroll">
    <div id="The hottest day of the summer so far was drawing to a close and a drowsy silence lay over the large, square houses of Privet Drive.”</div>
</div>
            
#container {
    border: 3px solid black;
    border-radius: 5px;
    overflow: hidden;
}

#text {
    transform: translateX(100%);
    
    animation: scroll-animation 30s linear infinite;
    }
    
    @keyframes scroll-animation {
    from {
        transform: translateX(100%);
    }
    to {
        transform: translateX(-100%);
    }
