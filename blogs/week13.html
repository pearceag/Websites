This week outside of class I learned about the video tag in html. This tag allows you to embed a video in your website. This a very helpful tool for websites where you want to show your user a tutorial or something cool through a video. You can specify certain attributes (width, height, autoplay, loop, controls, etc.). Below is the code and implementation of this tag. 

            
    <video autoplay="" loop="" controls="" width="640" height="480">
    <source type="video/mp4" src="https://endtest-videos.s3-us-west-2.amazonaws.com/documentation/endtest_data_driven_testing_csv.mp4">
    </video>
