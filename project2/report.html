Endpoint Explanation

'/report.html': takes you to this page

'/playlist': selects all items in the playlist

'/playlist/artist/:artist': selects all the playlist items with the same artist

'/playlist/genre/:genre': selects all the playlist items with the same genre

'/playlist/album/:album': selects all the playlist items with the same album

'/playlist/songName/:songName: selects all the playlist items with the same song name

post '/playlist/: inserts new item to the playlist

patch '/playlist/:id': updates the playlist item indicated by the id param

delete '/playlist/:id: deletes item in playlist at indicated by the id param
